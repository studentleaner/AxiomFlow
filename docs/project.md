# Project Format

When creating a new repository, utilizing `axiomflow init <project_name>` yields a pre-scaffolded generic execution boundary:

- **`registry/`**: Houses declarative boundaries directly readable securely by Axiom.
- **`workflows/`**: Handles localized standard logic wrappers to execute pipelines correctly.
- **`templates/`**: Hosts presets defined exclusively independent of active ContextFlow logic.

A project is functionally just structural logic—AxiomFlow commands orchestrate operations around these explicit containers cleanly.
