# Overview

**AxiomStudio** is the visual Integrated Development Environment (IDE) built specifically for the Axiom suite. 

It provides engineers, prompt authors, and architects with a strictly decoupled dashboard to author workflows, inspect registries, configure backend context mechanisms and execute graph topologies without writing raw initialization scripts.

## The Axiom Stack
1. **Axiom**: The schema definitions and prompt compiler.
2. **ContextFlow**: The unified execution engine orchestrating flows.
3. **AxiomFlow**: The developer CLI wrapping execution loops.
4. **AxiomStudio**: The graphic configurator unifying interactions.

> [!IMPORTANT]
> AxiomStudio acts exclusively as an interface configuration wrapper.
